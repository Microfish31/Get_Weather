# Get_Weather

說明:  
 輸入縣市  
 輸入鄉鎮市區  

 顯示 天氣狀況敘述  
 顯示 均溫圖表    

設計:  
資料來源 : 中央氣象局開放資料平台 (https://opendata.cwb.gov.tw/devManual/insrtuction)  
將資料下載至電腦(json格式)，透過python 解析，說明天氣並且畫圖呈現出來。
